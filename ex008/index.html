Aqui vão ter a principais formatações

Negrito

Eu posso usar "b" para ficar em Negrito.
Ou eu posso usar "strong", para ficar em Negrito.
Sendo respectivamente, sendo Não Semântica, e Semântica.

Itálico/Ênfase

Para colocar em Itálico, eu posso colocar a tag "i",
Ou então, "ctrl + shift + p", selecionar "wrap with abbreviation", e digitar "i + enter", ai a frase irá ficar em itálico.
Ou também, usando "em", para ficar em itálico.
Sendo respectivamente, Não Semântica e Semântica.

Marcado

Agora para marcar um texto, se usa "mark" para deixar o texto marcado.
Para mudar a cor da marcação, se usa o mesmo termo "mark" mas dentro dele se coloca a tag "style= + background-color + a cor desejada", mas isso é Css.
Caso queira mudar a cor padrão da marcação, tem q ir na parte <head>, cria uma tag <style> dentro dessa tag, se coloca outra "mark" + "espaço" + "{" pula de linha e coloca background-color e a cor.
Não coloca < e >, e lembrar de fechar a tag "style", mas já se fecha por padrão.

Tamanho texto

Para texto grande, se usa "big".
E texto pequeno, se usa "small".
Porém, não se usa a tag "big".

Sublinhado

Para sublinhar, se usa "del".

Inserido

Para mostrar que um texto foi inserido, se usa a tag "ins".

Texto Sobrescrito

Eu não sei o que é sobrescritooi, mas se usa a tag "sup".

Texto Subscrito

Eu também não sei o que é subscritotchau, mas a tag é "sub".
